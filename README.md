# Yngve Nykås Portfolio

Small single-page portfolio built with React, TypeScript, and Vite. The app uses hash-based navigation to switch between About, Portfolio, Résumé, and Contact sections without page reloads.

- Light/dark theme toggle and English/Norwegian language switch
- Tech/creative skill highlights with iconography
- Responsive layout with reusable nav, toggler, and section components

## Run it locally

```sh
npm install
npm run dev
```

Useful scripts:

- `npm run build` – production build to `dist/`
- `npm run preview` – serve the built site locally
- `npm run lint` – run ESLint over the project
